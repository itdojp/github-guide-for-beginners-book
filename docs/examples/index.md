# 実習サンプル（examples/）

このセクションでは、本書の実習で利用できる **サンプルファイル（テンプレ・雛形）** をまとめます。

サンプルはリポジトリの `examples/` に配置しています。

- `examples/`：https://github.com/{{ site.repository }}/tree/main/examples

## 収録内容

### Docs-as-Code（文書運用）サンプル

- 推奨ディレクトリ構成（`docs/`）
- 文書テンプレ（ADR/議事録/設計/Runbook）
- Issue/PR のサンプル文面

該当パス：`examples/docs-as-code/`

## 使い方（コピーして使う）

1. `examples/` から必要なフォルダを選びます。
2. 自分のリポジトリにコピーします。
3. 目的に合わせて、目次（`docs/index.md`）やテンプレの項目を調整します。
