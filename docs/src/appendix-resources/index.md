# 付録C 学習リソースと参考文献

GitとGitHubをさらに深く学ぶための厳選されたリソースを紹介します。

## 公式ドキュメント・チュートリアル

### Git公式

- **[Pro Git Book](https://git-scm.com/book/ja/v2)**
  - 無料のオンライン書籍
  - 日本語版も利用可能
  - Git の基礎から上級テクニックまで網羅

- **[Git Reference](https://git-scm.com/docs)**
  - 公式リファレンスドキュメント
  - 全コマンドの詳細説明
  - 英語版のみ

- **[Git Tutorial](https://git-scm.com/docs/gittutorial)**
  - 公式チュートリアル
  - コマンドライン操作の基礎

### GitHub公式

- **[GitHub Docs](https://docs.github.com/ja)**
  - 日本語対応の公式ドキュメント
  - 基本機能から高度な機能まで

- **[GitHub Skills](https://skills.github.com/)**
  - インタラクティブな学習コース
  - 実際のリポジトリで学習

- **[GitHub Learning Lab](https://lab.github.com/)**
  - 実践的なプロジェクトベース学習
  - ボットによる自動フィードバック
