# docs/index.md（文書ポータル）テンプレ

このファイルは、`docs/` 配下の文書を探すための入口（ポータル）です。  
README は最短導線に絞り、詳細本文は docs/ に寄せる運用を推奨します。

## 文書の種類

- 意思決定ログ（ADR）：`docs/adr/`
- 議事録：`docs/meeting-notes/`
- 設計書：`docs/design/`
- 運用手順（Runbook）：`docs/runbook/`
- テンプレ：`docs/templates/`

## 運用ルール（最小セット）

- 文書の追加・更新は Pull Request でレビューする
- 「決める・やる」は Issue に残し、「最新版の本文」は docs/ に残す
- 廃止した文書は削除ではなく「廃止」と明記し、後継へのリンクを置く

## 主要リンク

- テンプレ：`docs/templates/`
- 変更管理：Issues / Pull Requests
