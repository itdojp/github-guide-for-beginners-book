# Docs-as-Code サンプル

このサンプルは、GitHub を「ドキュメント管理・ナレッジ基盤」として運用するための最小セットです。

## 何が入っているか

- `docs/`：推奨ディレクトリ構成の例
  - `index.md`：ポータル（目次）
  - `templates/`：文書テンプレ（ADR/議事録/設計/Runbook）
  - `adr/` / `meeting-notes/` / `design/` / `runbook/`：サンプル文書
- `sample-issues/`：Issue/PR のサンプル文面

## 使い方（コピーして使う）

1. 自分のリポジトリに `docs/` を作ります。
2. このサンプルの `docs/` をコピーします。
3. 自分の目的に合わせて `docs/index.md` のリンクとテンプレ項目を調整します。
