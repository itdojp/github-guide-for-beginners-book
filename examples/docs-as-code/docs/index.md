# docs/（ポータル）

この `docs/index.md` は、リポジトリ内の文書を探すための入口（ポータル）です。  
README には最短導線だけを書き、詳細はここに集約する方針を推奨します。

## 文書の種類

- 意思決定ログ（ADR）：`docs/adr/`
- 議事録：`docs/meeting-notes/`
- 設計書：`docs/design/`
- 運用手順（Runbook）：`docs/runbook/`
- テンプレ：`docs/templates/`

## 使い方（運用ルールの最小セット）

- 文書の追加・更新は Pull Request でレビューします。
- 「決める・やる」は Issue に残し、「最新版の本文」は docs/ に残します。
- 廃止した文書は削除ではなく「廃止」と明記し、後継へのリンクを置きます。
