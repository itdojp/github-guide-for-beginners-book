# サンプルIssue（文書変更）

## タイトル例

`docs: ADRテンプレを追加する`

## 本文例

### 背景

- 文書の書き方が人によって違い、レビュー観点が揃わない。

### 目的

- ADR の最低限のフォーマットを用意し、意思決定の記録を残しやすくする。

### スコープ

- `docs/templates/adr-template.md` を追加する
- `docs/index.md` にテンプレへのリンクを追加する

### 完了条件（DoD）

- テンプレの項目が過不足なく、初学者でも埋められる
- docs/index.md からテンプレへ辿れる

### 補足

- 既存文書の書き換えは本Issueでは行わない（別Issueで扱う）
